Part Two

Link to dashboard using unique data. I used zoning data for munipalities in Wake County, NC. I created a histogram that displays building height by zone and building density is shown using width of bars in histogram. Using this dashboard, the user can view all zones as well as filter by particular zone(s) by selecting bars in the histogram. The user can also view differences in building height and density by zone in Wake County.
